# Rabit: Burrows for the Agentic Web

Welcome to the official Rabit project burrow. This is a Bun.js monorepo containing the specification, reference implementations, and tools.

## What is Rabit?

Rabit is a specification for publishing content that both humans and AI agents can navigate reliably. It defines a simple convention - a manifest file called `.burrow.json` - that tells visitors what's in your content space and how to traverse it.

## Getting Started

1. Read the [README](README.md) for an introduction
2. Review the [Specification](docs/rabit-spec-draft-2026-01-12.md) for technical details
3. Check out the [Examples](examples/) for practical implementations
4. Use the [Client](packages/rabit-client/) to traverse burrows programmatically
5. Try the [OpenCode Plugin](packages/opencode-rabit/) to integrate with AI tools

## Development

```bash
# Install dependencies
bun install

# Build all packages
bun run build

# Run tests
bun test
```
