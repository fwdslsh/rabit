# Frequently Asked Questions

## General

### What is Acme Platform?

Acme Platform is a comprehensive solution for building and deploying modern applications.

### Is Acme Platform free?

Acme offers a free tier for small projects. See our [pricing page](https://acme.example.com/pricing) for details.

## Technical

### What databases are supported?

Acme supports PostgreSQL, MySQL, and SQLite out of the box.

### Can I self-host Acme?

Yes! See the [Installation Guide](./guides/installation.md) for self-hosting instructions.

### How do I get API access?

Generate an API token from your dashboard at https://app.acme.example.com/settings/tokens.

## Support

### How do I report a bug?

Open an issue on our GitHub repository or email support@acme.example.com.

### Where can I get help?

- GitHub Discussions
- Discord community
- Stack Overflow (tag: `acme-platform`)
